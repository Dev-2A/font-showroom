# 🔤 Font ShowRoom

Google Fonts를 탐색하고, 자간·행간·굵기·크기를 슬라이더로 실시간 조절하며
한글/영문/숫자 샘플 텍스트를 비교할 수 있는 폰트 미리보기 쇼룸입니다.

> 🚧 개발 중입니다.

## 기술 스택

- React + Vite
- Google Fonts API
- Tailwind CSS

## 주요 기능

- 🔍 Google Fonts 검색 및 필터링
- 🎛️ 글꼴 크기 · 굵기 · 자간 · 행간 실시간 슬라이더 조절
- 🇰🇷 한글 / 🇺🇸 영문 / 🔢 숫자 샘플 텍스트 비교
- ⭐ 즐겨찾기 저장 (localStorage)
- 🌐 백엔드 없이 브라우저만으로 동작
- 📦 에어갭 환경 대응 가능
